# WeixinMPSaaS 搜索能力接入指南

两步完成配置搜索功能！
## 配置项

1.  准备好下列信息, 用于生成搜索服务功能 
   1. 开发者ID(AppID)
   2. 开发者密码(AppSecret)
   3. 消息加解密密钥(EncodingAESKey)
   4. 令牌(Token)
2.  配置 : IP白名单 
3.  配置 : 服务器地址(URL)
