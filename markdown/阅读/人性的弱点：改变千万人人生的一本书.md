# 人性的弱点：改变千万人人生的一本书

> 《人性的弱点》以人性具有的一般弱点为切入点，内容涉及人际交往、心态调节、维系家庭、安排工作和合理用钱等诸多方面。在书中，卡耐基为读者提供了如何与他人相处、如何得到他人的认同、如何回避人性的弱点、如何战胜人性的弱点的种种方法，并提出了应对各类弱点的有效策略。
卡耐基在书中总结出：“一个人在事业上的成功，有15％归结于他的专业知识，另外85％归结于他表达思想、领导他人和唤起他人热情的能力”。
